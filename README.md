# Pixel Starter

一个像素风格的 UI 组件库，适合 Web3 和 AI 项目。

## 像素组件

提供一系列像素艺术风格的 UI 组件：

- PixelButton - 像素按钮
- PixelCard - 像素卡片
- PixelBadge - 像素徽章
- PixelInput - 像素输入框
- PixelProgressBar - 像素进度条
- PixelTerminal - 像素终端
- 以及更多组件...

## 特点

- 复古像素设计
- 支持深色/浅色主题
- 响应式布局
- 基于 Tailwind CSS

## 快速开始

```bash
git clone https://github.com/iceberg211/pixel-starter.git
cd pixel-starter
pnpm install
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看效果。

## 自定义

修改 `app/globals.css` 来调整颜色和样式。
